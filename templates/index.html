Welcome to Bruno Filippone's Django REST Web Service Sample
running on Google App Engine

 

Web Service API

 

User Registration URL: api/registration/

Sample POST Request:

{ "username": "Bruno", "email": "bruno.filippone@b-fil.com", "password": "my_secret_password" }

 

Messaging URL: api/messaging/ (Requires Basic Authentication)

Sample GET Request:

api/messaging/?to_user=Bruno

Sample POST Request:

{ "to_user": "God", "message": "Hi God!" }
